# Public Assets

Place your logo file here. Supported formats:
- `logo.svg` (recommended for scalability)
- `logo.png`
- `logo.jpg`

The logo will automatically appear in:
- Header (next to "BlockPay" text)
- Footer (next to "BlockPay" text)

If you have a different filename, update the `src` attribute in:
- `components/layout/Header.tsx`
- `components/layout/Footer.tsx`
